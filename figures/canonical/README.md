# Canonical Figures

This directory is reserved for versioned reference figures that should survive
ordinary demo reruns. Use it for images that are cited in reviews, handouts, or
release notes and that need a stable visual baseline.

Before adding a canonical figure, record:

- the command or notebook used to generate it;
- the source data, random seed, and important parameter values;
- the expected qualitative features students should notice;
- whether it is meant to match a TeX figure, a Python demo, or a Mathematica
  companion.

Generated scratch figures belong in `figures/`, not here.
